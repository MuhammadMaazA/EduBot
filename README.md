# MI2US
Repository for the projects using Social Robots and LLM for multicultural integration

## Visualising GUI
1. Create a .venv folder, activate the virtual environment, and install the requirements.
   ``` 
   python -m venv .venv
   .venv\Scripts\activate
   python install -r requirements.txt
   ```

2. Run the program.
   ```
   python interface/main_program.py
   ```
